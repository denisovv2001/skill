Программа <main.c>
	Инструкция пользователя для решения квадратного уравнения.

	Файл main.c - консольное приложение написанное на языке программирования C. 
	Позволяет решить квадратное уравнение aX^2 + bX + c = 0.
	Для решения программе необходимо предоставить три коэффициента a, b и c, 
	после ввода данных программа предоставит ответ на задачу (без решения).


	Для запуска можно установить среду разработки Code::Blocks
		После установки открываем файл main.c с помощью Code::Blocks.
		Находим на панели Code::Blocks кнопку Build and run (сборка и запуск), нажимаем на неё или нажимаем клавишу f9 на клавиатуре.
	[Устанавливать не обязательно Code::Blocks. Можно любую другую среду разработки позволяющей выполнять программу на языке программирования Си]

	Задача нахождения корней решается с использованием следующих средств программирования:
		1) условные операторы (if [если], else if [иначе если], else [иначе])

		2) типы данных:
			а) int (целочисленные)
			б) double (числа с плавающей запятой)
			в) struct (структура)

		3) input/output - набор функций, обеспечивающих ввод-вывод

		4) Заголовочные файлы стандартных библиотек языка программирования С:
			а) stdio.h - ввод-вывод информации
			б) locale.h - для отображения русского языка на экран
			в) math.h - для взятия квадратного корня при подсчёте корней
